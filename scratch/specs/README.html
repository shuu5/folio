§4. Reading Order

新規 reader (人間 / AI agent) は以下の順序で読む SHOULD。

  1. constitution.html — folio の 12 不変原則 (前提知識、 全 spec の基底)
  2. rules.html — Layer 1 consumer project に適用する規範 MUST (markup / naming / Mandatory Actions、 spec edit 開始前必読)
  3. folio-self-spec.html — folio Layer 0 framework 自身の動作仕様 (harness / plugin / bindings)
  4. relations.html — spec 間関係性表現規約 (JSON-LD vocabulary、 spec 編集時に <head> JSON-LD を書く際必読)

spec edit を行う AI agent は最低限 (1) + (2) を必読 MUST、 関係性宣言を変更する場合は (4) も MUST。 (3) は folio framework の動作変更を行う場合のみ必読。
