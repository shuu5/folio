scratch/ — 試作・工事用箱

v0.4.2-draft · Phase X2 試作段階の作業場所

本 dir は folio framework が試作段階にある期間の 工事用一時的箱。 試作・調査・要望整理・実装試行は すべてここで行う。 完成後 (Phase X3) に正式 location (architecture/) に移植、 scratch/ は撤去予定。

§1. Purpose

folio framework は self-application paradox (自身を縛る rule で自身を書けない) を抱えている。 試作段階では folio rule に縛られず自由に書ける場所として scratch/ を採用する。

§2. Contents

現在 scratch/ 配下にあるもの (root から移動):

  • constitution.html — 12 不変原則 (編集禁止)
  • rules.html — Layer 1 向け universal rules (当面編集禁止)
  • folio-self-spec.html — folio 自身の architecture spec (当面編集禁止)
  • decisions/ — ADR cluster
  • assets/mermaid.min.js — vendor

将来 (試作の進行に応じて) 追加予定:

  • research/ — user 要望 + 業界調査 (HTML 形式)
  • prototype/ — 試作 plugin の draft
  • test/ — 試作の test 実行ログ

§3. Rules

  • scratch/ 内は folio rule に縛られない。 markup 正規性、 命名規約、 declarative form 等の rule は適用しない。 自由に書く。
  • 例外: 既存 file (constitution.html、 rules.html、 folio-self-spec.html、 decisions/ 配下) は **編集禁止** (移動された frozen 資産)。
  • 新規 file / dir 作成は自由。 思考 memo / 一時 work は柔軟に。

§4. Lifecycle

  • 現在 (Phase X2): 試作・調査の作業場所として active。
  • 完成後 (Phase X3): 試作 plugin を .claude-plugin/ に正式移植、 folio 自身の architecture spec を architecture/ に移植。 scratch/ は撤去 or archive。
